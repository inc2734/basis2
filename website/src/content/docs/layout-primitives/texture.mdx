## Props

### <Badge>HTML</Badge> -texture:\{variation\} / <Badge>JSX</Badge> texture=\{variation\}

Set any texture.

`Dots` / `offset-dots` / `grid` / `horizontal-stripe` / `vertical-stripe` / `checker-pattern`

<Tabs items={['HTML', 'JSX', 'Preview']}>
	<Tabs.Tab>
		```html
		<div data-unitone-layout="texture -texture:offset-dots">
			...
		</div>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		```jsx
		import { Texture } from '@inc2734/unitone-css';

		<Texture texture="offset-dots">
			...
		</Texture>
		```
	</Tabs.Tab>
	<Tabs.Tab>
		<PreviewConainer>
			<Texture texture="offset-dots">
				<Decorator padding={1}>
					<p>Lorem Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. 				</Decorator>
			</Texture>

	</Tabs.Tab>
</Tabs>
