# Get started

## Install

```shell
$ npm install --save-dev @inc2734/unitone-css
```

### Using Sass

```scss
@use 'PATH/TO/node_modules/@inc2734/unitone-css/src/app';
```

### Using CSS

```html
<link rel="stylesheet" href="/node_modules/@inc2734/unitone-css/dist/app.css" />
```
