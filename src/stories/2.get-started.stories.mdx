# Get started

## Install

```shell
$ npm install @inc2734/basis2
```

### Using Sass

```scss
@import 'PATH/TO/node_modules/@inc2734/basis2/src/app';
```

### Using CSS

```html
<link rel="stylesheet" href="/node_modules/@inc2734/basis2/dist/app.css" />
```
