### Customizing gap

```html
<div data-unitone-layout="stack -gap:1">
  ...
</div>
```

### Using negative gap

```html
<div data-unitone-layout="stack -gap:1 -negative">
  ...
</div>
```

### Using divider

stripe / underline / bordered

```html
<div data-unitone-layout="stack -divider:stripe">
  <div>
    ...
  </div>
  <div>
    ...
  </div>
</div>
```
